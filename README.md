# FinAgent: Agentic AI for Household Nutrition and Budgeting

FinAgent is an agentic AI system that jointly optimizes household meal planning and budgets using:

- Linear programming for cost-effective nutrient planning
- Price-aware adaptation to market fluctuations
- Multi-agent reasoning with diversity objectives

## Repository Structure
- `data/` : CSV datasets
- `docs/` : Dataset documentation, data card, system overview
- `examples/` : Demo notebook (Google Colab ready)

## Getting Started
1. Clone this repo.
2. Open `examples/demo_finagent.ipynb` in Google Colab.
3. Run cells sequentially to see sample recommendations.
