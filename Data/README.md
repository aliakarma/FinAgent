# FinAgent Datasets

This folder contains all datasets required for the FinAgent project.

## Datasets

1. **financial_data.csv**  
   - 101 rows, 8 columns  
   - Fields: user_id, monthly_income, rent, utilities, transport, education, healthcare, savings_target  

2. **food_composition.csv**  
   - 351 rows, 7 columns  
   - Fields: food_id, food_name, calories_per_100g, protein_g, vitamin_d_mcg, iron_mg, halal  

3. **food_prices.csv**  
   - 351 rows, 4 columns  
   - Fields: food_id, store, price_per_kg, date  

4. **nutrition_requirements.csv**  
   - 501 rows, 7 columns  
   - Fields: person_id, age, gender, calories_kcal, protein_g, vitamin_d_mcg, iron_mg  

All datasets are synthetic or aggregated from publicly available sources for research purposes.
